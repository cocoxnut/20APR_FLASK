# Installation
- just clone this repo
```
$ git clone https://github.com/cocoxnut/20APR_FLASK
```
- open repo directory
```
$ cd 20APR_FLASK
```
- create virtualenv & activate virtualenv 

```
$ python3 -m venv venv
$ . . /venv/bin/activate
```
- install dependences
```
$ pip install -r related.txt
```
- run project
```
$ export FLASK_APP=class
$ flask run
```
